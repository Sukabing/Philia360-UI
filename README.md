# Philia360-UI 资源包

引用 betmc 的动态背景效果/合掌

## 📥 下载与安装

1. 在 [Releases] 页面下载最新版本的 `.mcpack` 文件
2. 在 Minecraft 中导入资源包并激活
3. 确保将其置于全局资源包首位以确保最佳效果
